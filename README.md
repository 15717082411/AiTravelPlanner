# AiTravelPlanner

一个 Web 版的 AI 旅行规划师项目的基础骨架。当前仅搭建了目录结构，后续将根据具体需求逐步完善前后端能力与 CI/CD。

## 目录结构

- `frontend/` 前端应用（Web UI）
  - `public/` 静态资源
  - `src/components/` 组件
  - `src/pages/` 页面
  - `src/hooks/` 自定义 Hooks
  - `src/styles/` 样式
  - `src/utils/` 工具函数
  - `src/assets/` 资源文件
- `backend/` 后端服务（API）
  - `src/routes/` 路由定义
  - `src/controllers/` 控制器
  - `src/services/` 业务服务
  - `src/middleware/` 中间件
  - `src/config/` 配置
  - `src/models/` 数据模型
  - `src/utils/` 工具函数
- `docs/` 项目文档
- `tests/` 测试用例
- `scripts/` 自动化脚本

所有空目录通过 `.gitkeep` 文件被跟踪。
